== Description ==
This plugin enables a subscription bar which can be used to increase the number of subscribers to your mailing list. 
Features:
1. The bar can be used to subscribe users via any of the mailing service.
2. This plugin provides customization of: 
                                              a) Background Color
                                              b) Text
                                              c) Text Color
                                              d) Button Color
                                              e) Button Text Color
3.  Provides options to display the bar on either top or bottom of the page.
4.  Input the form code from the respective mailing service in the form input area.

== Installation ==
1. Download Subscription Bar plugin via WordPress.org plugin repository or by uploading the files to your server.
2. Activate the plugin.
3. On activation, go to  appearance--> customize.
4. Customize through Subscription Bar and Subscription Bar Display.
